# Dataset Overview

## Dataset Composition

The dataset consists of synchronized multimodal data segments extracted from
real-world video sources.

### Modalities

- **Video**: speaker-centered clips
- **Audio**: high-quality speech segments
- **Text**: multilingual transcriptions and translations

### Languages

- Amis
- Puyuma
- Paiwan
- Sasiyat
- Mandarin Chinese

### Segment Design

Each data segment is indexed by a unified timestamp and includes:

- Video clip
- Corresponding audio
- Source-language transcript
- Target-language translation 

This design ensures seamless integration into downstream intelligent applications.
