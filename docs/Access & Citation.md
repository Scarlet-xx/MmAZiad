# Access & Citation

## 📌 Dataset Access

The Multilingual Multimodal Intelligent Application Dataset is publicly available through the following website:

👉 https://scarlet-xx.github.io/MmAZiad/

The repository includes:
- Multimodal data samples (video, audio, text)
- Structured dataset organization
- Demo interface for visualization
- Statistical analysis results

---

## 📜 License & Usage

This dataset is released for **academic research purposes only**.

Users are allowed to:
- Use the dataset for research and educational purposes
- Conduct experiments on multilingual and multimodal learning tasks
- Cite the dataset in academic publications

Users are NOT allowed to:
- Use the dataset for commercial purposes without permission
- Redistribute the dataset without proper attribution
- Modify and republish the dataset as a new dataset
